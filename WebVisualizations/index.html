Summary: Latitude vs. X

The project looks at how distance from the equator impacts the weather by visualising four key weather components of cities across the world. With a random list of geographic coordinates, Citypy Python library was used to determine cities of varying distance from the equator. The OpenWeatherMap API was utilised to create a representative model of weather across world cities.

Visualisations of the dataset were created via Matplotlib. Scatterplots showcased the relationship between the latitude and each of the weather components - temperature , humidity, cloudiness and wind speed.

This site includes the dataset and its visualisations, including any correlations revealed.
